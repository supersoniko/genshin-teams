# Genshin-teams frontend

## Development Local

### Install dependencies
`$ npm i`

### Environment variables
`$ cp .env-example .env.local`

### Start local develop server
`$ npm run develop`

### Debugging

#### IntelliJ IDEA

You can easily debug this application with the intelliJ IDEA debugger.
Follow the steps of the official Node.js debugger [documentation of jetbrains](https://www.jetbrains.com/help/idea/running-and-debugging-node-js.html).

For the options use the following settings:

- Application parameters: `node_modules/gatsby/dist/bin/gatsby develop`
